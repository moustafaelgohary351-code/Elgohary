نبذة تعريفية 🌟

أنا مبرمج ومطور واجهات مستخدم شغوف، أمتلك القدرة على بناء تطبيقات ومواقع ويب ذكية وتفاعلية باستخدام أحدث التقنيات البرمجية، وأتحدى الظروف للتطوير من الهاتف المحمول.

المهارات البرمجية 🛠️

• بناء هيكلية المواقع باستخدام HTML5

• تنسيق وتصميم المواقع وجعلها متجاوبة باستخدام CSS3

• إضافة المنطق والذكاء البرمجي باستخدام JavaScript (DOM Manipulation)

• إدارة المشاريع ورفعها باستخدام Git & GitHub

معرض أعمالي 🚀

• تطبيق العداد الشرطي الذكي: تطبيق يغير الألوان ديناميكياً حسب تفاعل المستخدم.

• تطبيق إدارة المهام المتكامل: تطبيق يسمح بإنشاء وحذف المهام اليومية من العدم تلقائياً.
